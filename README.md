# AURORA LUXE — 極光金框 Slot（試玩版）

5 軸 × 4 排、14 條支付線的拉霸試玩 Demo。

## 玩法
- 固定下注 **2000**，按中央 ⟳ 旋轉。
- **黃金金框（Golden Frame）** 隨機落在盤面；只要金框落在**任一條中獎連線**上，即立即觸發**單一回合最大獎 20000**（= 10× 下注），本回合結束並發放。
- 百搭符號 **A** 可替代所有普通符號。
- 長期 RTP ≈ **97.4%**，平均每 **10.3** 轉觸發一次最大獎。

## 操作
- ⚡ 快速、▶ 自動、＋／－（本機固定注，僅提示）。
- 左上 ☰ 選單：遊戲說明 / 賠付表、模擬 RTP（20 萬轉）、重置餘額。
